      "pizzaName": "Pepperoni Pizza",

      "cookingMethod": "Baked"

🎯 Objectives:

    🔍 Parse the JSON object to extract pizza information.
    📋 Display the details in a user-friendly format.
    ⚠️ Handle potential parsing errors gracefully.
